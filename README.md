## Getting Started


### Prerequisites

How to download via Npm:
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Svxy/Useful-Fun-Python-scripts.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
